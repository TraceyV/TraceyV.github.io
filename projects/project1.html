    • 3 Point Perspective Art

     About


    The project consists of having to create a 3 Point Perspective by first sketching in paper and digitalizing it on Adobe Illustrator.

     Date


    April 2024

     Software used

     Process & Challenges


    I brainstormed ideas on what kind of illustration would I create using the 3 point perspective on paper. I decided to use buildings for the project and took my time digitalizing it in Illustrator using the 3D tool on the app itself.

    The challenge lies on doing a 3 point perspective starting from above and going down.


     Solution & Outcome


    I made sure that the illustration made sense in a way that if you were looking from above, you would see things trully shrink like the windows and the buildings.
